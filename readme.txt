Steps for working with git and GitHub:
Step 1 - Create a repository in GitHub
Step 2 - Copy the clone link and clone it down onto your local machine (git clone yourgithubrepo.git)
Step 3 - touch a new file
Step 4 - run code . to open your project in VS Code
Step 5 - add some code
Step 6 - It's time to ACP!  Start this process by running git add filename
Step 7 - commit your changes by running git commit -m "some message about your changes"
Step 8 - push your changes to GitHub by running git push origin master (edited)
